PL/SQL Developer

Job Title: PL/SQL Developer 
Primary skills: SQL, PL/SQL development and testing
Mode of work: Work from Office
Experience: 4 - 7 Years
About the job
We are looking for a PL/SQL Developer with 4-7 years of experience in designing, developing, and testing PL/SQL code. The role involves writing complex SQL queries, stored procedures, and database tuning to optimize performance. The candidate should have strong knowledge of Oracle SQL Server and multiple data sources to create data transformation and translation solutions.

Responsibilities:
•	Design and develop PL/SQL code such as Packages, Stored Procedures, Functions, Triggers, Collections, and Types based on requirements.
•	Write complex SQL queries and stored procedures for database management.
•	Optimize queries and resolve key performance issues.
•	Extract and connect data from different sources to create dashboards.
•	Interact with IT and business teams to coordinate project cycles.
•	Research source data and design solutions to fix ongoing issues.
•	Ensure best practices in report development, testing, and scheduling.

Requirements
Must have:

•	4-7 years of experience in designing, developing, testing, and deploying PL/SQL solutions.
•	Strong knowledge of database tuning, query optimization, and performance troubleshooting.
•	Experience with Oracle SQL Server and multiple data sources.
•	Proficient in SQL programming for data transformation and translation.
•	Experience in ETL tools and Unix knowledge is an added advantage.
•	Oracle certification is a plus.
