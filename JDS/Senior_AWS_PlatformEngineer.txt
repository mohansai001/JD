Senior AWS Platform Engineer
Role Overview:
As a Senior AWS Platform Engineer, you’ll design, implement, and maintain highly available, scalable, and secure AWS platforms. You’ll focus on building infrastructure solutions, automating deployments, and improving platform performance.

Key Responsibilities:

Design and implement AWS infrastructure to support scalable applications using EC2, RDS, Lambda, and other AWS services.
Build and optimize CI/CD pipelines and automation frameworks for fast, reliable deployments.
Implement infrastructure as code with CloudFormation or Terraform for consistent, reproducible environments.
Set up monitoring, alerting, and logging tools (CloudWatch, ELK stack) to ensure platform stability and security.
Troubleshoot complex platform issues, perform root-cause analysis, and implement permanent fixes.
Mentor junior engineers and collaborate with developers to align platform capabilities with application needs.
Key Skills:

In-depth knowledge of AWS services and infrastructure architecture.
Advanced experience with CI/CD tools and infrastructure as code (Terraform, CloudFormation).
Strong scripting skills in Python, Bash, or PowerShell.
Knowledge of monitoring and alerting tools, including CloudWatch, ELK, and Grafana.
Experience with containerization (ECS, EKS, Docker).
